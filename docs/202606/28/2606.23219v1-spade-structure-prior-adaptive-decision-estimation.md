## 摘要
物理结构先验（如守恒定律、哈密顿形式和对称性）在正确时能改进科学机器学习，但指定错误时则会降低预测性能。现有方法通常强制执行选定的结构或调节软惩罚，但缺乏校准规则来决定是否施加先验、施加强度、使用哪个先验，或候选定律的哪个子集成立。我们提出SPADE（结构先验自适应决策估计），这是一个闭式框架，将问题视为无约束估计量中违反结构部分的收缩。SPADE使用一个精确的规范检验和一个估计量：检验决定先验是否被数据支持，斯坦无偏的詹姆斯-斯坦收缩以$O(σ^2/n)$的oracle保证设置强制执行强度，且仅当检验认证时才承诺硬先验。同样的检验能够实现一致的嵌套结构选择，并在非嵌套约束族中对子集发现进行Benjamini-Hochberg控制。在线性子空间先验、水库守恒律以及Duffing动力学上的非线性哈密顿先验中，SPADE追踪oracle，击败神经网络基线，将正确先验遗憾从10.3%降至2.6%，以1/71的求解次数匹配交叉验证，以100%的准确率选择正确结构，并以受控的虚假松弛恢复部分定律。

## Abstract
Physical-structure priors such as conservation laws, Hamiltonian forms, and symmetries can improve scientific machine learning when correct, but can degrade predictions when misspecified. Existing methods usually enforce a chosen structure or tune a soft penalty, without a calibrated rule for deciding whether to impose a prior, how strongly to impose it, which prior to use, or which subset of candidate laws holds. We introduce SPADE, Structure-Prior Adaptive Decision Estimation, a closed-form framework that treats this problem as shrinkage of the structure-violating block of an unconstrained estimator. SPADE uses one exact specification test and one estimand: the test decides whether the prior is supported by data, Stein-unbiased James-Stein shrinkage sets the enforcement strength with an $O(σ^2/n)$ oracle guarantee, and a gate commits to the hard prior only when the test certifies it. The same test yields consistent nested structure selection and Benjamini-Hochberg control for subset discovery in non-nested constraint families. Across a linear-subspace prior, a reservoir conservation law, and a nonlinear Hamiltonian prior on Duffing dynamics, SPADE tracks the oracle, beats a neural-network baseline, reduces correct-prior regret from $10.3\%$ to $2.6\%$, matches cross-validation with $1/71$ of the solves, selects the correct structure with $100\%$ accuracy, and recovers partial laws with controlled false relaxation.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **研究动机**：在科学机器学习中，物理结构先验（如守恒定律、哈密顿形式、对称性）被广泛应用于提升样本效率和泛化能力。然而，当这些先验与真实系统不完全匹配（错误指定）时，硬性施加结构会引入偏差，导致模型性能退化，甚至比无约束模型更差。现有方法要么强制使用选定的先验（如哈密顿神经网络），要么通过软惩罚调整约束强度（如物理信息神经网络），但缺乏一个统计上校准的决策规则来回答四个关键问题：是否施加先验、施加强度、选择哪个先验、以及哪些候选定律成立。
- **整体含义**：SPADE首次将结构先验的施加问题转化为一个无约束估计中违反先验参数块的收缩决策，通过一个闭式框架同时解决了上述四个问题，实现了数据驱动的自适应先验使用。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：物理结构先验若在参数空间中是线性子空间 \(V\)（例如守恒律、对称性），则违反先验的分量位于正交补 \(V^\perp\) 中。无约束估计量（OLS）在该块上的估计 \(x\) 的范数直接反映了先验的错配程度。对 \(x\) 进行收缩，等价于在完全保留（自由模型）和完全剔除（硬先验模型）之间自适应调节。
- **关键技术细节**：
  - **违反块收缩**：定义自适应估计量 \(\hat{\theta}_{\text{adapt}}(t) = P_V \hat{\theta}_{\text{free}} + t x\)，其中 \(t \in [0,1]\)。最优收缩系数为 \(t^* = \frac{\|\mu\|^2}{\|\mu\|^2 + B}\)，其中 \(\mu = P^\perp \theta^*\)，\(B = \sigma^2 \operatorname{tr} C\)。该系数是James-Stein和Stein无偏风险估计的目标。
  - **无调参插件**：使用Stein无偏风险估计得到闭式插件：\(\hat{t} = \left(1 - \frac{\hat{\sigma}^2 \operatorname{tr} C}{\|x\|^2}\right)_+\)，其中 \(\hat{\sigma}^2 = \frac{\|z - \Phi \hat{\theta}_{\text{free}}\|^2}{n-p}\)。对于病态设计（如水库特征映射），采用留出校准分裂估计 \(t^*\)。
  - **精确规范检验**：在零假设 \(H_0: \delta = 0\)（先验正确）下，检验统计量 \(F = \frac{x^\top C^{-1} x}{m \hat{\sigma}^2} \sim F_{m, n-p}\)，精确控制第一类错误为 \(\alpha\)。当检验接受时，直接返回硬先验估计；拒绝时，返回收缩估计。这种“测试门控”避免了正确先验时自适应收缩的额外代价。
  - **嵌套结构选择**：对于嵌套先验族 \(V_1 \subset V_2 \subset \cdots \subset V_K\)，依次检验每个先验，选择第一个未被拒绝的结构，实现一致选择。
  - **非嵌套子集发现**：对 \(K\) 个候选约束（每个对应一个 \(V^\perp\) 块），计算每个约束的 \(p\)-值，应用Benjamini-Hochberg过程控制错误发现率（FDR），从而发现哪些约束成立。
