## 摘要
大型语言模型（LLM）为科学发现提供了有前景的途径，但其在符号回归中的应用通常受到低效搜索策略和粗略反馈信号的限制。当前方法通常使用标量指标（例如全局均方误差）引导LLM，但无法识别所提议方程中哪些成分驱动了性能或导致了误差。我们引入了影响引导的符号回归（IGSR），该方法将方程发现构建为一个迭代的两步过程，结合了多样化的项生成与严格的选择：LLM为线性模型生成候选基函数 ψ_j(x)，然后使用细粒度影响分数 Δ_j 对其进行评估。这些分数量化了每个项对泛化准确率的边际贡献，从而实现影响引导的剪枝过程，系统地细化模型结构。将此机制集成到蒙特卡洛树搜索（MCTS）中，可以在导航组合搜索空间的同时，平衡对新函数形式的探索与对高影响成分的利用。我们在多个基准测试上展示了IGSR的有效性，包括LLM-SRBench、药理学PKPD模型、流行病学模拟和真实世界基因组数据。值得注意的是，我们通过使用高维生物学数据集的案例研究验证了该框架的真正发现能力，其中IGSR识别出DNA甲基化与RNA聚合酶II暂停之间的新型关系；该假设随后通过湿实验得到了支持。

## Abstract
Large Language Models (LLMs) offer a promising avenue for scientific discovery, yet their application to symbolic regression is often constrained by inefficient search strategies and coarse feedback signals. Current methods typically guide LLMs using scalar metrics (e.g., global Mean Squared Error), which fail to identify which components of a proposed equation are driving performance or causing error. We introduce \textit{Influence-Guided Symbolic Regression} (IGSR), a method that frames equation discovery as an iterative two-step process combining diverse term generation with rigorous selection: an LLM generates candidate basis functions $ψ_j(\mathbf{x})$ for a linear model, which are then evaluated using granular influence scores $Δ_j$. These scores quantify each term's marginal contribution to generalization accuracy, enabling an influence-guided pruning process that systematically refines the model structure. Integrating this mechanism into a Monte Carlo Tree Search (MCTS) enables navigating the combinatorial search space while balancing exploration of novel functional forms with exploitation of high-influence components. We demonstrate IGSR's effectiveness on a diverse suite of benchmarks, including LLM-SRBench, pharmacological PKPD models, an epidemiological simulation, and real-world genomic data. Notably, we validate the framework's capacity for genuine discovery in a case study using a high-dimensional biological dataset, in which IGSR identified a novel relationship between DNA methylation and RNA Polymerase II pausing; a hypothesis that was subsequently supported via wet-lab experimentation.

## 2. 方法论
### 核心思想
- **IGSR**（Influence-Guided Symbolic Regression）将方程发现构建为“提议-剪枝”迭代循环，并嵌入蒙特卡洛树搜索（MCTS）中。
- 关键创新：使用**每项影响分数（Δ_j）**，量化每个候选基函数 ψ_j(x) 对验证集MSE的边际贡献（若将该项权重置零导致的MSE增加），实现细粒度信用分配。

### 关键技术细节
- **模型形式**：目标为稀疏线性模型 f(x)=∑w_j ψ_j(x)，其中 ψ_j(x) 由LLM提议。
- **提议阶段**：LLM根据任务描述、当前活跃项、最佳方程和历史反馈（包括之前哪些项被保留/丢弃及其影响）生成约5个（首轮10个）候选基函数。
- **评估阶段**：将所有候选项合并，用OLS拟合线性模型，然后在验证集上计算每个项的影响分数 Δ_j = MSE(删去该项) - MSE(完整模型)（不重新拟合，保持其他权重固定）。
- **剪枝阶段**：默认采用**确定性剪枝**：按 Δ_j 排序，保留前K=6个项。也可用**IGSR-Agent**变体，由第二个LLM基于Δ_j和语义推理进行剪枝。
- **搜索策略**：将上述循环嵌入MCTS。每个节点代表一个方程状态（一组基函数及其权重），子节点通过随机LLM提议生成，使用UCT选择，奖励为负验证MSE。默认使用“启发式MCTS”（无rollout，直接回传节点奖励），总预算30次扩展。

- **实际科学发现**：在RNA Pol II暂停案例中，IGSR发现DNA甲基化抑制转录暂停的新假设，并通过5-氮杂胞苷处理实验验证（Wilcoxon p值极小），证明了从符号模型到湿实验验证的完整闭环。
- IGSR具有亚线性计算扩展性：特征数从10增至263时，运行时间仅增加约25%。
- 跨种子稳定性：在已知PKPD方程上，IGSR在25个种子中高度一致地恢复核心机制项。

## 7. 优点
- **细粒度反馈机制**：Δ_j直接衡量每个项对泛化的贡献，使搜索从全局损失优化转向结构化组件优化，大幅提升效率。
- **解耦生成与选择**：LLM负责创意提议，确定性/基于Δ_j的剪枝确保统计严谨，避免LLM幻觉保留噪声项。
- **高效高维处理**：通过LLM先验和影响剪枝，避免组合爆炸，在高维（263特征）数据集上仍能快速收敛。
- **真正的科学验证**：不仅是数值评估，还通过湿实验确认了AI生成的新假设，展示了“AI生成假设→实验验证”的完整科研范式。
- **模块化与鲁棒性**：支持确定性/代理剪枝、MCTS/线性迭代、多种LLM骨干，对提示变化、无关特征等表现稳定。

## 8. 不足与局限
- **LLM先验限制**：发现能力受限于LLM预训练分布中的函数形式；若所需函数完全不在其知识范围内，系统可能无法提出。
- **线性模型假设**：IGSR假设结构为 f(x)=∑w_j ψ_j(x)；虽然可以优化项内常数（IGSR-TLO变体），但发现完全嵌套的非线性动力系统仍有挑战。
- **计算成本和提示敏感性**：尽管比传统符号回归高效，但LLM API调用仍占主要成本；提示细节影响性能（简化提示在高维数据更便宜但精度略降）。
- **符号恢复局限**：在材料科学领域（LLM-SRBench），IGSR对特定域内函数（如Arrhenius型速率）的恢复不如D3或LLM-SR，倾向于用线性展开近似。
- **优化目标单一**：当前仅优化验证MSE，未显式纳入稀疏性或复杂度惩罚（如项数、算子复杂度），可能影响Pareto前沿的探索。
- **重复性风险**：虽然跨种子稳定，但科学发现案例中甲基化假设在70-80%种子中恢复，仍有部分种子未检出，显示轻微的随机变异性。
