## 摘要
大语言模型为科学发现提供了有前景的途径，但其在符号回归中的应用常受限于低效的搜索策略和粗糙的反馈信号。当前方法通常使用标量指标（如全局均方误差）引导大语言模型，这无法识别所提出方程中哪些成分驱动性能或导致错误。我们提出影响引导的符号回归（IGSR），该方法将方程发现构建为迭代的两步过程：结合多样化项生成与严格筛选——大语言模型为线性模型生成候选基函数ψ_j(x)，然后使用细粒度影响分数Δ_j进行评估。这些分数量化每个项对泛化准确率的边际贡献，从而实现影响引导的剪枝过程，系统性地优化模型结构。将该机制集成到蒙特卡洛树搜索中，能够在平衡探索新函数形式与利用高影响成分的同时，导航组合搜索空间。我们在多样化的基准测试集上展示了IGSR的有效性，包括LLM-SRBench、药理学PKPD模型、流行病学模拟以及真实世界的基因组数据。值得注意的是，我们在一个高维生物学数据集的案例研究中验证了该框架进行真正发现的能力——IGSR识别出DNA甲基化与RNA聚合酶II暂停之间的新颖关系；该假设随后通过湿实验得到了支持。

## Abstract
Large Language Models (LLMs) offer a promising avenue for scientific discovery, yet their application to symbolic regression is often constrained by inefficient search strategies and coarse feedback signals. Current methods typically guide LLMs using scalar metrics (e.g., global Mean Squared Error), which fail to identify which components of a proposed equation are driving performance or causing error. We introduce \textit{Influence-Guided Symbolic Regression} (IGSR), a method that frames equation discovery as an iterative two-step process combining diverse term generation with rigorous selection: an LLM generates candidate basis functions $ψ_j(\mathbf{x})$ for a linear model, which are then evaluated using granular influence scores $Δ_j$. These scores quantify each term's marginal contribution to generalization accuracy, enabling an influence-guided pruning process that systematically refines the model structure. Integrating this mechanism into a Monte Carlo Tree Search (MCTS) enables navigating the combinatorial search space while balancing exploration of novel functional forms with exploitation of high-influence components. We demonstrate IGSR's effectiveness on a diverse suite of benchmarks, including LLM-SRBench, pharmacological PKPD models, an epidemiological simulation, and real-world genomic data. Notably, we validate the framework's capacity for genuine discovery in a case study using a high-dimensional biological dataset, in which IGSR identified a novel relationship between DNA methylation and RNA Polymerase II pausing; a hypothesis that was subsequently supported via wet-lab experimentation.

## 2. 论文提出的方法论

### 核心思想
- 将方程发现建模为**提出-剪枝循环**（Propose-and-Prune Cycle），嵌入**蒙特卡洛树搜索（MCTS）** 框架。
- LLM负责生成候选基函数 ψ_j(x)，然后计算每个基函数的**影响分数 Δ_j**，量化其对验证集泛化精度的边际贡献。
- 基于Δ_j进行**确定式剪枝**（默认）或**LLM代理剪枝**（IGSR-Agent变体），保留高影响项，丢弃低影响项。
- 通过MCTS平衡探索新函数形式与利用高影响成分，导航组合搜索空间。

### 关键技术细节
- **目标模型形式**：f(x) = Σ w_j ψ_j(x)，线性于非线性基函数，保持全局可解释性。
- **影响分数Δ_j定义**：对于验证集，将项j的权重w_j置零（其他权重固定）时，验证MSE的增加量。Δ_j = MSE_{-j} - MSE_full。Δ_j越大，项越重要；Δ_j ≈ 0表示冗余。
- **计算方式**：先在全候选集上拟合OLS模型，得到权重w和预测，然后对每个项j，通过减去φ_{val,j} w_j快速计算移除后的MSE，无需重新拟合。高效且适用于迭代。
- **确定式剪枝**：按Δ_j降序排名，保留top K项（默认K=6）。多输出时跨输出取最大值。
- **MCTS集成**：每个节点代表一个方程状态（当前项集）。从父节点通过提出-剪枝循环生成多个子节点（默认5个），使用UCT公式（探索常数c=√2）选择节点。奖励为负验证MSE，直接回传（Heuristic MCTS，无深层 rollout）。
- **历史记忆**：每次循环的保留/丢弃项及MSE变化记录到历史缓冲，供LLM在后续迭代中上下文学习。

### 算法流程（文字说明）
1. **初始化**：设置根节点（空项集），总扩展预算（默认30）。
2. **MCTS迭代**：
   - **选择**：使用UCT选择叶子节点。
   - **扩展**：从该节点执行提出-剪枝循环（默认生成5个子节点）。
     - **提出**：LLM根据问题描述、当前项集、最佳方程、历史记录，生成约5个（首轮10个）新候选基函数。
     - **评估**：合并所有候选基函数，在训练集拟合OLS线性模型，计算验证集上每项Δ_j。
     - **剪枝**：按Δ_j保留top K项（默认6），构成新方程状态。
   - **奖励与回传**：新节点的验证MSE作为奖励，回传更新祖先节点统计。
3. **终止**：达到预算后，返回验证MSE最小的方程作为最终结果。

  - 消融实验：4种变体（有无MCTS、有无影响反馈），每种25种子。
  - 灵敏度分析：9种不同LLM（包括开源和闭源）在6个数据集上×25种子（LLaMA为10种子）。
  - 鲁棒性实验：添加5/75/150个不相关特征，25种子。
  - 合成模型：5种修改结构×25种子。
  - RNA聚合酶案例研究：独立生物学重复、特征子集随机化（10种子）。
  - 此外还进行了MCTS rollout深度、项数目、提示变化等超参数探索实验。
- **公平性与客观性**：所有方法使用相同LLM后端（GPT-4o或GPT-4o-mini），保持相同token预算（LLM-SRBench限定300k tokens），复杂度匹配（项数限制为6）。统计方法正确（Student's t检验、重复测量ANOVA）。实验设计覆盖广泛场景，结果稳定且可复现。

## 6. 论文的主要结论与发现

1. **性能优势**：IGSR在6个基准数据集中（5/6）取得最低MSE，在LLM-SRBench上平均排名第一（ID和OOD预测指标）。消融实验证实细粒度影响分数和MCTS均显著提升效果。
2. **高维扩展性**：输入特征数从10增加到263，运行时间仅增加约25%，避免组合爆炸，适用于高维生物数据。
3. **跨种子稳定性**：在已知真实方程的癌症数据上，IGSR发现的核心项（如药物相互作用、Gompertz增长）在25种子中高度一致（前4项频率84-100%），微观变化仅体现在非线性项的近似方式。
4. **真实科学发现**：在RNA聚合酶II暂停案例中，IGSR识别出DNA甲基化与暂停强度的负相关假设，并通过5-azacytidine处理的湿实验证实：去甲基化显著增加暂停频率和强度（Wilcoxon p≈10^{-52}和10^{-33}）。
5. **消融验证**：移除影响反馈或MCTS均导致NMSE显著上升（从0.000787分别升至0.293和0.626），说明两者协同重要。

## 7. 优点

- **方法创新**：将逐项影响分数引入LLM驱动的符号回归，实现结构化信用分配，将生成与选择解耦。
- **搜索效率**：结合MCTS系统探索，避免局部最优，收敛速度显著优于遗传规划方法（GPLearn）。
- **高维友好**：子线性扩展，可处理数百特征，远超传统符号回归方法。
- **可解释性与验证性**：输出简洁符号方程，影响分数提供每项解释。案例研究完成“AI生成假设→湿实验验证”闭环，体现真正科学发现能力。
- **鲁棒性实验充分**：涵盖多种LLM、噪声特征、合成结构、独立生物学重复等压力测试，结果一致。

## 8. 不足与局限

- **依赖LLM先验**：发现能力受限于LLM预训练分布。若所需函数形式不在LLM知识范围内，系统可能无法提出。
- **模型形式限制**：当前假设f(x)=Σ w_j ψ_j(x)，虽允许非线性基函数，但无法表达深度嵌套的动态系统（如递归神经网络形式的ODE）。IGSR-TLO变体可优化常数，但整体仍是线性组合。
- **缺乏显式稀疏性惩罚**：当前仅通过保留K项控制复杂度，未将项数或复杂度直接纳入奖励函数，可能错过帕累托最优解。
- **计算成本**：LLM API调用是主要瓶颈，在资源受限环境下可能不适用。虽然总运行时间可接受，但仍高于传统非LLM方法。
- **潜在过拟合风险**：虽然嵌套验证分析表明无显著过拟合，但搜索过程仍可能对验证集过度适应，需在更复杂数据上进一步验证。
- **特定领域偏差**：案例研究集中在生物转录领域，泛化到其他科学领域（如物理、化学）需更多测试。
- **实验覆盖**：虽然实验丰富，但部分对比方法（如SINDy）在高维数据集上因组合爆炸无法完成，降低了直接可比性。
